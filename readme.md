# Speller

Speller is a simple browser extension that lets you play Tic-Tac-Toe directly from your browser popup.  
It is designed to be lightweight, fast, and easy to play anytime without opening a separate website.

The game supports both player vs player and player vs AI modes, making it useful for quick breaks while browsing.

## Features

- **Human vs Human Mode**  
  Play Tic-Tac-Toe with a friend on the same device.

- **Human vs Robot Mode**  
  Challenge an AI opponent powered by the Minimax algorithm.

- **Smart AI Moves**  
  The robot calculates the best possible move, making the game challenging.

- **Win Detection**  
  Automatically detects winning patterns and highlights the winning cells.

- **Draw Detection**  
  The game detects when the board is full and declares a draw.

- **Restart Game**  
  Restart the match anytime with the restart button.

- **Lightweight Popup Game**  
  Runs directly inside the browser extension popup without loading a full page.

### Install Manually (Chrome / Chromium)

1. Download the latest release from the link above.
2. Extract the ZIP file.
3. Open your browser and go to:


4. Enable **Developer Mode** (top right).
5. Click **Load unpacked**.
6. Select the extracted extension folder.

The **Speller** extension will now appear in your browser toolbar.

## How to Play

1. Click the **Speller extension icon** in your browser.
2. Choose a game mode:
   - Human vs Human
   - Human vs Robot
3. Click any cell on the board to make your move.
4. The game automatically detects wins and draws.
5. Click **Restart** to start a new game.
